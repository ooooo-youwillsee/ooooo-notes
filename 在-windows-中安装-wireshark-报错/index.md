# 在 Windows 中安装 Wireshark 报错



&gt; **win10安装wireshark经常报“KB2999226 和 KB3118401”**

1. install wireshark
2. open installation directory, **manually install vc_redist.x64.exe** by double click
3. reinstall wireshark
