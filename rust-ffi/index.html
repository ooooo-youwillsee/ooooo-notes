实现一个简单的 ffi 绑定, 分为C函数调用和lib函数调用。

C 函数调用

函数是 C 内置的，无需任何额外处理。

use std::ffi::{c_char, CString};

extern {
    // 计算字符串长度
    fn strlen(ptr: *const c_char) -> usize;
}

fn main() {
    let s = CString::new("123").unwrap();
    let len = unsafe { strlen(s.as_ptr()) };
    println!("len: {}", len);
}
