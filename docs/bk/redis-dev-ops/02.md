## 1、预备

### 1、全局命令

- 查看所有的key

```shell script
  KEYS *
```

- 键总数
```shell script
  DBSIZE
```

- 检查键是否存在
```shell script
  EXISTS key
```

- 删除键
```shell script
  DEL key
```

- 键过期
```shell script
  EXPIRE key sencond
```

- ttl 返回过期时间
```shell script
  TTL key
```
\>0: 剩余过期时间

-1: 没有设置过期时间

-2: 键不存在

- key的数据结构类型
```shell script
  TYPE key
```

### 2、数据结构与内部编码

每种数据结构都有自己底层的内部编码实现，通过命令 `OBJECT ENCODING key` 来查看。

- string 内部编码： `raw`、`int`、`embstr`
- hash 内部编码： `ziplist`、`hashtable`
- list 内部编码： `ziplist`、`linkedlist`、`quicklist`
- set 内部编码： `intset`、`hashtable`
- zset 内部编码： `ziplist`、`skiplist`
