# 1、Kafka 是分布式流处理平台

Apache Kafka 是消息引擎系统，也是一个分布式流处理平台。

Kafka 的特性：

  • 提供一套 API 实现生产者和消费者。
  • 降低网络传输和磁盘存储开销。
  • 实现高伸缩性架构。
  • 可以实现端到端的精确一次处理语义。
